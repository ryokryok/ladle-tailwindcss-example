# Ladle (alt Storybook) with Tailwind CSS example

This repo is example for [Ladle](https://ladle.dev/).

## stack

aim to **fast** development.

- React
- TypeScript
- Tailwind CSS
- Ladle
- SWC Plugin
- Vitest
  - Happy DOM
  - React Testing Library

## CLI

```bash
# start ladle development mode
$ pnpm story
# build ladle as static site
$ pnpm build-story
# preview built ladle
$ pnpm preview-story
# run unit test
$ pnpm test
# output unit test coverage
$ pnpm coverage
```
